How to Get Mods for Gorilla Tag (PC Only)

Follow these steps to install mods on your PC version of Gorilla Tag using BepInEx!

What You Need

To get started with mods for Gorilla Tag on PC, make sure you have the following:

  • A PC with Steam and Gorilla Tag installed.
  • SteamVR setup if you're using a VR headset.
  • Basic file management knowledge (extracting files, navigating directories, etc.).

Step-by-Step Installation Guide

  1. Backup Your Game Files - Always create a backup of your game files in case something goes wrong.
  2. Download BepInEx - This is the modding framework you need for installing mods in Gorilla Tag.
    • Go to the official BepInEx GitHub page: Download BepInEx.
    • Make sure you choose the correct version (for Unity games) and download the zip file.
  3. Extract BepInEx Files - After downloading BepInEx, extract the files to your Gorilla Tag installation folder. This is typically located at: C:\Program Files (x86)\Steam\steamapps\common\Gorilla Tag.
    • Make sure all the files from the extracted BepInEx folder are placed in this directory (e.g., BepInEx, doorstop_config.ini, etc.).
  4. Download Mods - Find mods from trusted sources like Monke Mod, AKA, PC Modding group!
    • MonkeMod - Gorilla Tag Mods
  5. Install Mods - Once you've downloaded the mod, extract its contents into the BepInEx/plugins folder inside your Gorilla Tag directory.
    • The mod's DLL file should go into the plugins folder.
  6. Launch Gorilla Tag - Start the game via Steam, and you should see the mods active in-game.
