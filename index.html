How to Get Mods for Gorilla Tag (PC Only)

Follow these easy steps to install mods on your PC version of Gorilla Tag!

What You Need

Before you begin, make sure you have the following:

  • A PC with Steam and Gorilla Tag installed.
  • SteamVR setup if you're using a VR headset.
  • Basic file management skills (extracting files, navigating folders, etc.).

Step-by-Step Installation Guide

1. Backup Your Game Files

Always create a backup of your game files to avoid any issues.

2. Download BepInEx

You'll need BepInEx for modding. Download it from the official page:

Download BepInEx

3. Extract BepInEx Files

After downloading, extract the files into your Gorilla Tag directory. Typically, the path is:

C:\Program Files (x86)\Steam\steamapps\common\Gorilla Tag

4. Download Mods

Get mods from trusted sources. Here are some recommended modding groups:

  • MonkeMod - Gorilla Tag Mods

5. Install Mods

Place the downloaded mod's DLL file into the BepInEx/plugins folder inside the Gorilla Tag directory.

6. Launch Gorilla Tag

Once you've installed your mods, launch the game through Steam, and the mods should be active!
