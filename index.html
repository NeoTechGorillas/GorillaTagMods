How to Get Mods for Gorilla Tag (PC Only)

Enhance your Gorilla Tag experience with these simple modding steps.

What You Need

Make sure you have the following:

  • A PC with Steam and Gorilla Tag installed.
  • SteamVR set up if using a VR headset.
  • Basic file management skills.

Step-by-Step Guide

1. Backup Your Game Files

It's always a good idea to back up your game files before modding.

2. Download BepInEx

Download BepInEx from the official site:

Download BepInEx

3. Extract BepInEx Files

Extract the BepInEx files into your Gorilla Tag installation directory. Common paths:

C:\Program Files (x86)\Steam\steamapps\common\Gorilla Tag
C:\Program Files\Oculus\Software\Software\another-axiom-gorilla-tag

4. Download Mods

Find mods from trusted communities, like:

MonkeMod - Gorilla Tag Mods

5. Install Mods

Place the mod's DLL files in the BepInEx/plugins folder inside the Gorilla Tag directory.

6. Launch Gorilla Tag
