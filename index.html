How to Get Mods for Gorilla Tag (PC Only)

Install mods and enhance your Gorilla Tag experience with these simple steps!

What You Need

Before you begin, ensure you have the following:

  • A PC with Steam and Gorilla Tag installed.
  • SteamVR set up if using a VR headset.
  • Basic file management skills (extracting files, navigating folders, etc.).

Installation Guide

1. Backup Your Game Files

Always back up your game files to avoid losing any important data or mods.

2. Download BepInEx

BepInEx is essential for modding Gorilla Tag. Download the latest version from the official site:

Download BepInEx

3. Extract BepInEx Files

Extract the BepInEx files into your Gorilla Tag installation directory. Common paths:

C:\Program Files (x86)\Steam\steamapps\common\Gorilla Tag
C:\Program Files\Oculus\Software\Software\another-axiom-gorilla-tag

4. Download Mods

Get mods from trusted sources, like the MonkeMod community:

MonkeMod - Gorilla Tag Mods

5. Install Mods

Place the downloaded mod's DLL file into the BepInEx/plugins folder.

6. Launch Gorilla Tag

Launch the game through Steam, and your mods should now be active!
