How to Get Mods for Gorilla Tag

Enhance your Gorilla Tag experience with easy-to-follow modding steps.

What You Need

Before starting, make sure you have the following:

  • A PC with Steam and Gorilla Tag installed.
  • SteamVR setup if using a VR headset.
  • Basic file management skills.

Step-by-Step Guide

1. Backup Your Game Files

Always back up your game files to avoid any mishaps while modding.

2. Download BepInEx

BepInEx is a modding framework. Download it from the official site:

Download BepInEx

3. Extract BepInEx Files

Extract the downloaded BepInEx files into your Gorilla Tag directory:

C:\Program Files (x86)\Steam\steamapps\common\Gorilla Tag

Alternatively, if you installed it via Oculus, you can extract BepInEx files into:

C:\Program Files\Oculus\Software\Software\another-axiom-gorilla-tag

4. Download Mods

Get mods from trusted sources like:

MonkeMod - Gorilla Tag Mods

5. Install Mods

Place the mod's DLL files in the BepInEx/plugins folder within your Gorilla Tag directory.

6. Launch Gorilla Tag

After installing the mods, launch Gorilla Tag through Steam, and enjoy the new features!
